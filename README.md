# PUBLIC ART 👾

**[https://glaseagle.github.io/public-art/](https://glaseagle.github.io/public-art/)**

A terminal-aesthetic presentation environment for the Public Art Sharing Initiative. Built as a single HTML file with a CRT shader overlay, amber glow, and inline PDF slide viewer.

Press `Enter` to boot. Use `←` `→` arrow keys to navigate slides.
